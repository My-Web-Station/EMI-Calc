# EMI Calculator

It allows you to calaculate the monthly payment and interest amount based on principle amount and loan tenure. 

### Installation
For development, you will need Node.js installed on your environement. 

Install the dependencies and start the server.

```sh
$ git clone https://github.com/Gopigaj/EMI-Calculator.git
$ cd EMI-Calculator
$ npm install
$ npm run start
```

### Build with

- API :https://ftl-frontend-test.herokuapp.com/interest?amount=<amount>&numMonths=<numMonths>
- Slider :https://github.com/react-component/slider
- chart :https://github.com/toomuchdesign/react-minimal-pie-chart
